Overview
Stock Data Analysis & Prediction using Big Data Analytics and Machine Learning. This project leverages Hadoop ecosystem tools such as MapReduce, Hive, Sqoop, and PySpark to process and analyze large-scale stock market data from 2018–2023. It performs aggregation of key metrics like Open, Close, High, Low, and Volume on monthly and yearly levels, along with exploratory data analysis to identify trends and volatility patterns. Machine learning models including Linear Regression and Random Forest were implemented for stock price prediction, achieving high accuracy (R² ≈ 0.999). The system provides scalable, distributed data processing and delivers actionable insights for financial analysis and decision-making.

Objectives
1. Analyze historical stock market trends
2. Perform large-scale data processing using Hadoop ecosystem tools
3. Identify patterns, volatility, and performance metrics
4. Predict future stock prices using machine learning models

Dataset
-> Multi-stock dataset (12 companies)
-> Time period: 2018 – 2023
-> Features include:
     1.Open, Close, High, Low prices
     2. Volume
     3. Date / Time attributes
